Изменение контента на главной

Чтобы поменять картинку текст или название блоков в разделе новое нужно сделать такие шаги:
  • Перейти по ссылке ...
  • Открыть файл index.html
  • Нажать на кнопку изменить файл
  • Скролить до строки 53 и увидеть путь к картинке и текст
  • После именений сахраняем файл

Изменение контента на других страницах
